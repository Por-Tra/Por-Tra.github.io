RPG 2D

Description : Ce projet est un projet étudiant qui a été réaliser pendant mes années de lycée.
Il s'agit d'un jeu vidéo en 2D de type RPG dans un style Pokemon.
Le but du jeu est de parcourir 3 niveaux afin d'éliminer le boss de fin.

Langages : Python

Equipe : Travaillé en équipe de 2

Date : 2022-2023

Fonctionnalités : Déplacements, changement de niveau, animation, dialogues avec NPC, combat, changement de personnage

Mon rôle : Développeur

Compétences : Python, utilisation d'un IDE (pycharm), gestion de bibliothèque, utilisation d'outil de conceptionde niveau
